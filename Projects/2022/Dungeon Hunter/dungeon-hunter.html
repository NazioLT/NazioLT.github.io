Dungeon Hunter

Project summary

Project Type : School Project
Project dates : November 2022
Software used : Unity Engine 2021
Language used : C#
Primary Role : Programmer / Game Designer

Dungeon Hunter is a game where you play a knight and you have to clear dungeons from monsters that have taken over. This game was created in 3 weeks for a school project.
