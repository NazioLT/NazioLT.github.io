NConsole

Project summary

Project type : Tool
Project dates : May 2023 - Present
Software used : Unity Engine 2022
Language used : C#
Primary role : Tool programming

NConsole is an in-game procedural command console wich help you to easily debug your game.

The Tool

Features :

  • Display all Unity's logs.
  • Enter Commands.
  • Add your own commands.
  • Autocompletion.
  • Command history.
  • Theme Customization.

    • Commands :

  • Enum support.
  • Static methods.
  • Non-static methods.
  • Asynchronous methods.
  • Multiple execution types.

    • Demo

    Basic Usage :

    In your code :
  • Import the Nazio_LT.Tools.Console namespace.
  • Type [NCommand] above your method.
    • In Unity :
  • Go to your game, and create a console.
  • Go in console and type your command Name.

    • Use on all script instances :

    In your code :
  • Type [NCommand] above your method, and define the ExecutionMode to AllMonoBehaviourInstances.
    • In Unity :
  • For the test, we're going to add 2 scripts in the scene.
  • The Console Execute the methods of all scripts.
